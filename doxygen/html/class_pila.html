Detailed Description

template<typename T = int>
class Pila< T >

Plantilla de clase que implementa una pila con arreglo redimensionable.

Esta clase define una pila genérica que permite almacenar datos de cualquier tipo. Utiliza un arreglo dinámico que puede duplicar su capacidad cuando es necesario.

Template Parameters
TTipo de dato que almacenará la pila.
